## Installation

- `git clone https://github.com/tsakib360/get-your-court.git`
- `cd get-your-code/`
- `composer install`
- `cp .env.example .env`
- Update `.env` and set your database credentials
- `php artisan key:generate`
- `php artisan migrate`
- `npm install`
- `npm run dev`
- `php artisan serve`
